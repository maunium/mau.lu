mau.lu



Description

mau.lu is the frontend for the official instance of mau\Lu. mau\Lu is a simple URL shortening backend that supports custom short URLs and things like creating and shortening LMGTFY links easily.
Unshortening is just as easy as shortening: simply paste the short URL and press unshorten.
Please note that anyone can shorten links here. I'm not responsible for the targets of mau.lu short URLs.
If you want to disable previews of short URLs in chat apps, use the HTML redirect option.
